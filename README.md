Python LangChain
----------------
>Python LangChain beer style app using OpenAI.

Warning
-------
>Langchain is truly a ***piece of work!*** ;)

Install
-------
1. uv add langchain langchain-openai
2. uv add --dev ruff

Ruff
----
1. uvx ruff format
2. uvx ruff check

Run
---
1. uv run main.py [beer style]
