Python LangChain
----------------
>LangChain app using OpenAI.

Warning
-------
>Avoid this library at all costs!

Install
-------
1. uv add langchain openai
2. uv add --dev ruff

Ruff
----
1. uvx ruff format
2. uvx ruff check

Run
---
1. uv run main.py
