Python LangChain
----------------
>LangChain app using OpenAI.

Install
-------
1. uv add langchain
2. uv add langchain-openai
3. uv add langchain-community
4. uv add --dev ruff

Ruff
----
1. uvx ruff format
2. uvx ruff check

Run
---
1. uv run main.py
